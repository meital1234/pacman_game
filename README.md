- Control **Pacman** using the **arrow keys**.
- Pacman keeps moving in the last chosen direction ("auto-run").
- A **ghost** chases Pacman using a greedy distance-based algorithm.
- The ghost moves **slower** than Pacman.
- Pacman can use the **horizontal tunnel** to wrap from one side of the maze to the other.
- **Game Over** appears when the ghost catches Pacman.
- **YOU WIN!** appears when Pacman eats all dots in the maze.
